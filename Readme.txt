This Artefact is for Anomaly Detection System using Python. The Artefact can be divided in 4 Sections

1] IDS.ipynb - The base file is this one. All of the information needed for cleaning, pre-processing, and model training is contained in this file. Running this file produces every output that is desired.

2]exp.ipynb - This file is used for performing exploratory analysis on the chosen dataset.

3] Web Integration - The user selects the dataset, uploads it, and the model predicts the target variable and displays the metrics in a web-based application. This can be done using app.py
